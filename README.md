# React + Vite + Tailwind

Stack: **React**, **Vite**, **Tailwind CSS** (v4 via `@tailwindcss/vite`), JavaScript + JSX, ESLint.

Official React plugins:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) — Babel (or oxc with rolldown-vite) for Fast Refresh
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) — SWC for Fast Refresh

## React Compiler

Not enabled in this template (dev/build cost). To add it, see [React Compiler installation](https://react.dev/learn/react-compiler/installation).

## Scripts

- `npm run dev` — dev server
- `npm run build` — production build
- `npm run preview` — preview production build
- `npm run lint` — ESLint
